This project was created using data from "Stegonometrographia" by Melchias Uken. It was originally a way to send messages encoded as Latin poems. In this website it has been repurposed to work in reverse, Generating a poem based on a message input. Each letter in the input corresponds to a half line of Latin poetry. Usage: To generate a poem just type whatever alphabetical message you want into the text box and hit submit every two letters will generate a line of latin poetry
